# AUDITTRAIL.md

> Open standard for tamper-proof AI agent action logging and non-repudiation.

## Why This Matters

When an AI agent acts autonomously, organizations need irrefutable proof of what happened, when, and under whose authority.

## Quick Start

```bash
curl -O https://raw.githubusercontent.com/totalmarkdown/agent-md-specs/main/templates/AUDITTRAIL.template.md
mv AUDITTRAIL.template.md AUDITTRAIL.md
# Fill in your agent's configuration
```
